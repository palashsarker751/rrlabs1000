API Documentation

Welcome to the RRLabs API. Our API allows you to programmatically manage abandoned carts, trigger Gemini AI recovery sequences, and track your ROI.

Post

Incoming Webhook

Send abandoned cart data from your store to this endpoint to trigger an automated recovery sequence.

Endpoint URL Copy
https://api.rrlabs.online/v1/webhook/{{YOUR_SECRET_KEY}}

Request Body (JSON)

{
  "customer_name": "Rahat Karim",
  "email": "rahat@example.com",
  "phone": "+8801700000000",
  "cart_value": 149.50,
  "currency": "USD",
  "items": [
    { "id": "p1", "name": "Wireless Headphones", "price": 149.50 }
  ],
  "checkout_url": "https://yourstore.com/checkout/re-8842"
}

Authentication

All API requests must include your API Key in the request header.

Authorization
Bearer {{YOUR_API_KEY}}
